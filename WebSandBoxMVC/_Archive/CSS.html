Different Ways to Apply CSS

Inline Styles

Inline styles use the style attribute inside of an HTML tag. This is typically avoided, except in rare cases, because it isn't reusable. It uses semicolon-seperated, CSS property/value pairs in the attribute's value. Highest in terms of proximity and specificity but is the least reusable.

Internal Style

Internal style is defined inside the head of a page. It is more reusable, as the styling applies to the whole page. It is second in terms of proximity and specificity.

External Style

External styles are written in .css style sheets, which are brought into each page via a link in the head. This is best practice as it is reusable for the entire site. Lowest in terms of proximity and specificity.

Imported Style

Imported styling is done via an @import at the top of an external style sheet. It allows one style sheet to use styling from another one. This is rarely used, most typically when seperation of concerns is an issue or when using styling from a resource outside of the solution.

CSS Selectors

By Tag Name

This targets all elements of that tag's type, such a p, li, or h1

By Class

Called by adding the class attribute to an HTML tag. Can be used multiple times per page. Defined in the CSS with a period.

By ID

Called by adding the id attribute to an HTML tag. Used a maximum of once per page. Defined in the CSS with a hashtag.
